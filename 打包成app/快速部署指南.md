# 🚀 快速部署指南 - 3种方法

由于自动部署遇到权限问题，这里提供3种简单的手动部署方法：

---

## 方法一：Vercel Web部署（最简单，推荐）⭐⭐⭐⭐⭐

### 步骤：

1. **访问 Vercel 网站**
   - 打开浏览器访问：https://vercel.com/
   - 点击右上角 "Sign Up" 注册（可用GitHub账号登录）

2. **导入项目**
   - 登录后，点击 "Add New..." → "Project"
   - 选择 "Import Git Repository"
   - 如果项目在GitHub：授权GitHub并选择仓库
   - 如果项目不在GitHub：点击下方 "Import from Template" 旁边的 "Browse" 上传文件

3. **上传方式（推荐）**
   - 点击 "Deploy from GitHub" 下方的 "Import Third-Party Git Repository"
   - 或者直接将整个项目文件夹压缩成ZIP
   - 访问 https://vercel.com/new
   - 拖拽ZIP文件到页面

4. **配置项目**
   - Framework Preset: 选择 "Vite"
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Install Command: `npm install`

5. **部署**
   - 点击 "Deploy" 按钮
   - 等待2-3分钟构建完成
   - 完成！会得到一个网址，例如：`https://your-app.vercel.app`

## 方法二：Netlify 拖拽部署（超简单）⭐⭐⭐⭐⭐

### 步骤：

1. **构建项目**（如果还没构建）
   ```bash
   cd F:\AI编程\academic-timetable
   npm run build
   ```

2. **访问 Netlify Drop**
   - 打开浏览器：https://app.netlify.com/drop
   - 或访问 https://netlify.com → 登录 → Sites → "Add new site" → "Deploy manually"

3. **拖拽部署**
   - 直接将 `F:\AI编程\academic-timetable\dist` 文件夹
   - 拖拽到浏览器页面的虚线框中
   - 等待上传完成（1-2分钟）
   - 完成！会得到一个网址，例如：`https://random-name.netlify.app`

4. **自定义域名（可选）**
   - 在站点设置中可以修改二级域名
   - 例如改成：`https://my-timetable.netlify.app`

## 方法三：GitHub Pages 部署⭐⭐⭐⭐

### 步骤：

1. **构建项目**
   ```bash
   cd F:\AI编程\academic-timetable
   npm run build
   ```

2. **推送到 GitHub**
   - 如果还没有仓库，创建一个GitHub仓库
   - 将项目推送到GitHub

3. **配置 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source: 选择 "Deploy from a branch"
   - Branch: 选择 `main` 或 `master`
   - Folder: 选择 `/dist` 或 `/root`（需要先将dist内容提交）

4. **访问**
   - 等待1-2分钟
   - 网址：`https://你的用户名.github.io/仓库名/`

## 📱 推荐流程（最快5分钟）

### 使用 Netlify Drop（无需命令行）

1. 打开文件夹：`F:\AI编程\academic-timetable\dist`
   
2. 确认dist文件夹存在（如果不存在，运行 `npm run build`）

3. 访问：https://app.netlify.com/drop

4. 注册/登录 Netlify账号（可用GitHub登录）

5. 将整个 `dist` 文件夹拖拽到页面

6. 等待上传完成，获得网址！

7. 用手机访问这个网址：
   - Android Chrome: 菜单 → 添加到主屏幕
   - iOS Safari: 分享 → 添加到主屏幕

## 🆘 如果还是不行

### 最简单的本地测试方法：

1. **打开命令提示符（管理员权限）**
   - 按 `Win + X`
   - 选择"终端（管理员）"或"PowerShell（管理员）"

2. **运行预览服务器**
   ```bash
   cd F:\AI编程\academic-timetable
   npm run preview
   ```

3. **手机访问**
   - 查看终端显示的 Network 地址
   - 例如：`http://192.168.5.103:4173`
   - 用手机浏览器访问这个地址
   - 点击"添加到主屏幕"

**注意**：这种方法只在电脑开机且在同一WiFi下有效
