### ✅ 优势

## 🔧 自定义选项

### 修改应用名称
编辑 `public/manifest.json`:
```json
{
  "name": "我的课程表",
  "short_name": "课表"
}
```

### 更换应用图标
1. 准备一个512x512的PNG图片
2. 访问 https://www.pwabuilder.com/imageGenerator
3. 上传图片生成各尺寸图标
4. 下载并替换 `public/icons/` 目录中的文件
5. 重新构建部署

### 修改主题颜色
编辑 `public/manifest.json`:
```json
{
  "theme_color": "#你的颜色",
  "background_color": "#你的背景色"
}
```

## 🆘 常见问题

### Q1: 部署脚本运行失败？
**A**: 检查是否安装了Node.js，版本需要 >= 14
```bash
node -v  # 查看版本
```

### Q2: 手机无法安装PWA？
**A**: 
- 确保使用Chrome（Android）或Safari（iOS）
- 刷新页面重试
- 检查是否使用HTTPS（本地localhost除外）

### Q3: 离线后无法访问？
**A**: 
- 首次访问必须联网（用于缓存）
- 之后即可完全离线使用
- 如果无法离线，检查Service Worker是否启用

### Q4: 数据会丢失吗？
**A**: 
- 只要不清除浏览器数据或卸载应用，数据会一直保留
- 建议定期导出备份
- 换手机前记得导出数据

### Q5: 如何更新应用？
**A**: 
- 开发者重新部署后
- 用户下次打开应用时会自动检查更新
- 刷新页面即可获取最新版本

### Q6: 可以上架应用商店吗？
**A**: 
- PWA本身不能直接上架
- 可以使用Capacitor打包成原生APK后上架
- 参考 `PWA部署指南.md` 中的详细教程
