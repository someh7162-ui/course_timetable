✅ **2分钟完成** - 拖拽上传即可  
✅ **完全免费** - 无限流量  
✅ **自动HTTPS** - 安全访问  
✅ **全球CDN** - 访问速度快  

---

## 📋 详细步骤

### 第一步：确认构建文件存在

1. 打开文件夹：`F:\AI编程\academic-timetable`

2. 检查是否有 `dist` 文件夹
   - ✅ 如果有，跳到第二步
   - ❌ 如果没有，继续下面操作：

3. 打开命令提示符（Win + R，输入 `cmd`，回车）

4. 运行以下命令：
   ```bash
   cd F:\AI编程\academic-timetable
   npm run build
   ```

5. 等待构建完成（约10秒）

6. 确认 `dist` 文件夹已生成

---

### 第二步：访问 Netlify Drop

1. 打开浏览器

2. 访问：https://app.netlify.com/drop
   - 或访问：https://netlify.com
   - 然后点击 "Get started for free"

---

### 第三步：注册/登录账号

**首次使用需要注册：**

1. 点击页面上的 "Sign up" 或 "Get started"

2. 选择注册方式：
   - 推荐：用 GitHub 账号登录（一键授权）
   - 或：用邮箱注册

3. 完成注册/登录

---

### 第四步：上传部署

**如果在主页：**
1. 点击 "Add new site" → "Deploy manually"

**如果在 Drop 页面：**
1. 会看到一个虚线框，上面写着 "Drop your site folder here"

**开始上传：**

1. 打开文件管理器，定位到：`F:\AI编程\academic-timetable\dist`

2. 选中整个 `dist` 文件夹

3. 直接拖拽到浏览器的虚线框中

4. 等待上传（进度条显示）

5. 上传完成后，Netlify会自动部署

---

### 第五步：获取网址

部署完成后（约1-2分钟）：

1. 页面会自动跳转到站点详情

2. 顶部会显示网址，例如：
   ```
   https://random-name-123456.netlify.app
   ```

3. 点击网址，确认应用正常运行

---

### 第六步：自定义域名（可选）

1. 在站点详情页，点击 "Site settings"

2. 找到 "Site information" → "Change site name"

3. 输入你想要的名称，例如：`my-timetable`

4. 保存后，网址变为：
   ```
   https://my-timetable.netlify.app
   ```

---

### 第七步：安装到手机

**复制网址**，例如：
```
https://my-timetable.netlify.app
```

**在 Android 手机（Chrome）：**

1. 打开 Chrome 浏览器

2. 访问上面的网址

3. 等待页面完全加载

4. 点击右上角菜单 `⋮`（三个点）

5. 向下滚动，找到 "添加到主屏幕" 或 "安装应用"

6. 点击后，输入应用名称（或使用默认）

7. 点击 "添加" 或 "安装"

8. 完成！桌面会出现应用图标

**在 iPhone（Safari）：**

1. 打开 Safari 浏览器

2. 访问上面的网址

3. 点击底部中间的 "分享" 按钮 ↑

4. 向下滚动，找到 "添加到主屏幕"

5. 点击后，输入名称

6. 点击右上角 "添加"

7. 完成！桌面会出现图标

## 🆘 常见问题

### Q1: 上传后页面空白？
**A**: 
- 确保上传的是 `dist` 文件夹本身，不是dist里面的文件
- 检查dist文件夹中是否有 `index.html`

### Q2: 图标不显示？
**A**: 
- 首次访问后才会显示
- 刷新页面重试
- 或清除浏览器缓存后重新访问

### Q3: 无法离线访问？
**A**: 
- 首次访问必须联网（下载缓存）
- 访问一次后，关闭网络再试

### Q4: 手机找不到"添加到主屏幕"？
**A**: 
- Android: 确保使用Chrome浏览器
- iOS: 确保使用Safari浏览器
- 某些浏览器不支持PWA安装

### Q5: 网址太长记不住？
**A**: 
- 在Netlify设置中自定义站点名称
- 或生成二维码，手机扫码访问
